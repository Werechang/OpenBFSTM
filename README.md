# OpenBFSTM
## Overview
This program will be able to decode and encode
different kinds of audio files used in Super Mario Odyssey.

| File  | Decode  | Encode | File Versions | Compatibility Notes |
|-------|---------|--------|---------------|---------------------|
| BFSTM | ✅       | ⬜      | 6.1.0         |                     |
| BFGRP | ⬜       | ⬜      | 1.0.0         |                     |
| BFSAR | ⬜       | ⬜      | 2.4.0         |                     |
| BFSTP | ⬜       | ⬜      |               |                     |
| BFWAR | ⬜       | ⬜      |               |                     |
| BFWSD | ⬜       | ⬜      |               |                     |
| BFBNK | ⬜       | ⬜      |               |                     |
| BFSEQ | ⬜       | ⬜      |               |                     |
